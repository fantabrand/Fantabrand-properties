# Fantabrand Properties Ltd — Modern Luxury Real Estate Website

This is a Next.js website for **Fantabrand Properties Ltd**, a modern luxury real estate brand.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — Home
- `/about` — About Fantabrand Properties
- `/services` — Services
- `/properties` — Property listings
- `/properties/[slug]` — Single property detail page
- `/contact` — Contact form
